Code Examples

Basic Vector Operations

// Create and manipulate vectors
vec v1 = vec_new(3);
vec_set(v1, 0, 1.0);
vec_set(v1, 1, 2.0);
vec_set(v1, 2, 3.0);

// Vector addition
vec v2 = vec_add(v1, v1);
vec_print(v2);  // Output: [2.0, 4.0, 6.0]

Vector Expressions

// Complex vector expressions
expr e = expr_parse("v1 + v2 * 2");
vec result = expr_eval(e);
vec_print(result);

Installation

1. Clone the Repository

git clone https://github.com/yourusername/axl.git
cd axl

2. Build the Project

make

3. Run AXL

./axl

What you can do with AXL

Big Vectors

Work with large vectors efficiently and safely.

math.axl 
vec a = vec_new(1000);
vec_fill(a, 1.0);
vec b = vec_scale(a, 2.5);
vec_print(b);

Expression Engine

Parse and evaluate vector expressions on the fly.

expr.axl 
expr e = expr_parse("a + b * 2");
vec result = expr_eval(e);
vec_print(result);

CLI Ready

Run scripts and commands directly from your terminal.

stats.axl 
vec data = vec_from_array(arr, 100);
double mean = vec_mean(data);
printf("Mean: %f", mean);

Data Science Ready

Integrate with your data pipelines and perform fast vectorized computations.

cli.axl 
$ axl -e "sum([1,2,3,4])"
10

Extensible

Plug in your own C functions and extend the interpreter for your needs.
