Key features

Start in minutes

Tiny core

Hundreds of kilobytes compiled — fast cold starts, minimal attack surface, excellent for containers and edge.

Plain C

Portable, auditable, and straightforward to embed. No C++ runtime, no bindings required to get started.

Truly zero dependencies

Not even libc at runtime. Link statically, ship anywhere — from tiny ARM boards to multi‑socket servers.

Extreme performance

SIMD‑first kernels, cache‑aware layouts, batched IO. Built to keep hot paths hot and tail latency low.

Beyond‑memory datasets

Stream and index data that doesn’t fit in RAM. Predictable throughput under sustained load.

Production‑ready primitives

Crash‑safe persistence, vector‑optimized indexes, and simple APIs designed for real systems.

Build from source

Embed the library
#include "axl.h" int main(void) { axl_db *db = axl_open("data.axl", AXL_CREATE_IF_MISSING); axl_index_opts opts = axl_index_opts_default(); axl_add_vector(db, "users", (const float*)buf, 128, user_id, &opts); axl_search(db, "users", (const float*)query, 128, 10, results); axl_close(db); return 0; }
